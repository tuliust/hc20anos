# Design Tokens

## Cores

- Verde HC escuro: `#0d1a0f`
- Verde fundo profundo: `#080f08`
- Verde card: `#141f14`
- Verde institucional: `#2d6a4f`
- Verde claro: `#74c69d`
- Texto verde: `#7a9a7a`
- Off-white: `#f0ebe0`
- Dourado/acento: `#c9a84c`
- Vermelho erro: `#c0392b`

## Breakpoints

- Mobile: 320–767px
- Tablet: 768–1023px
- Desktop: 1024px+

## Componentes

- `Btn`: variantes `primary`, `outline`, `ghost`, `gold`, `danger`.
- `StatusBadge`: status de pagamento, moderação, roles e check-in.
- `EmptyState`, `ErrorState`, `LoadingState`, `PermissionState`.
- Cards: fundo escuro, borda verde translúcida.
- Modais: overlay escuro, conteúdo mobile-first.

## Tipografia

- Interface: system/DM Sans quando disponível.
- Títulos editoriais: `Playfair Display` quando disponível.
- Metadados: fonte monoespaçada via `font-mono`.
